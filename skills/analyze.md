# Token Analysis Skill

## Purpose
Provide deep analysis of specific tokens when requested by user.

## Analysis Framework

### [1] Fundamental Analysis
- Token utility and use case
- Team and development activity
- Community engagement metrics
- Partnership and ecosystem integration
- Tokenomics (supply, distribution, inflation)

### [2] On-Chain Metrics
- Unique wallet addresses (24h / 7d / 30d)
- Exchange inflows/outflows
- Holder distribution changes
- Large transaction frequency
- Smart money wallet tracking

### [3] Technical Analysis
- Support and resistance levels
- Moving averages (20/50/200 EMA)
- RSI and MACD indicators
- Volume profile analysis
- Chart pattern recognition

### [4] Risk Assessment
- Contract audit status
- Liquidity lock status
- Owner privileges analysis
- Tax and fee structure
- Slippage and impact estimates

## Confidence Grades
- **A**: High conviction setup with multiple confirmations
- **B**: Decent setup with some confirmation
- **C**: Low conviction, insufficient data

## Recommendation Types
- **STRONG BUY**: A-grade with excellent risk/reward
- **BUY**: B-grade with acceptable risk/reward
- **WATCH**: C-grade, monitor for better entry
- **AVOID**: Negative fundamentals or high risk
